### üöÄ Featured Algorithms & Development

#### üå≤ [ISCL: Individual Tree Detection Algorithm](https://ergin-portfolio2.vercel.app/)
*In Development (2025)*
* **Problem:** Traditional CHM-based and DL-based tree detection algorithms fail in overlapping multi-layered canopies.
* **Solution:** A novel algorithm utilizing inter-slice based clustering and geometric-graph fitting to isolate individual trees from high-density LiDAR point clouds.
* **Tech:** Python and R bindings for cloud processing.
