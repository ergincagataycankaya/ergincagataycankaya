### üë®‚Äçüíª About Me

I am a **Geospatial Data Scientist** and **PhD Researcher** specializing in the development of automated algorithms for forest inventory. My work bridges the gap between **forest biometrics** and **computer vision**, utilizing terrestrial/UAV LiDAR and multispectral imagery to create scalable measurement solutions.

Currently, I am designing **deep learning pipelines (U-Net)** for tree segmentation and developing the **ISCL algorithm** for individual tree detection in complex canopy structures.

* **Focus:** LiDAR Point Cloud Processing, Computer Vision, Forest Growth Modeling (FVS).
* **Infrastructure:** Cloud-native geospatial analysis (Azure/Docker) and Big Data processing (~10TB datasets).

### üöÄ Featured Algorithms & Development

#### üå≤ [ISCL: Individual Tree Detection Algorithm](https://ergin-portfolio2.vercel.app/)
*In Development (2025)*
* **Problem:** Traditional watershed algorithms fail in overlapping multi-layered canopies.
* **Solution:** A novel algorithm utilizing voxel-based clustering and geometric primitive fitting to isolate individual trees from high-density LiDAR point clouds.
* **Tech:** Python, C++ bindings for cloud processing.

#### üìä [National Forest Inventory Dashboard](https://tuoe.me/)
*Lead Developer & Architect*
* **Description:** A full-stack analytical platform visualizing multi-year national inventory data for Turkey.
* **Architecture:** Containerized **R Shiny** application hosted on **Microsoft Azure**.
* **Key Features:**
    * Distributed spatial querying for real-time analytics.
    * Interactive visualizations using `Plotly` and `Leaflet`.
    * Automated ETL pipelines for data updates.
